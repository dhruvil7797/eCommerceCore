# Introduction 
eCommerce Application's backend server running on .NET Core

# Features
- Add product
- Search product
- Add product to cart
- Modify product quantity
- Payment module
- Order product
- Add comments for product (Only allowed if you have bought that product in the past)
- View comments
- View order history
